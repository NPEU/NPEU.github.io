This pattern uses width for the marker rather than positioning, because IE doesn't move the marker if the window is resized.

Note that the tinted background is not part of the pattern - it's there to illustrate that the pattern has a transparent background.

175 out of 1020 recruited.
1020 out of 1020 recruited.
Show code: 
<div class="c-meter">
        <meter min="0" max="1020" value="175">
            <span style="width: calc(175 / 1020 * 100%);">175 out of 1020 recruited.</span>
        </meter>
        <div aria-hidden="true" class="c-meter__marker" style="width: calc(175 / 1020 * 100%);" data-value="175"></div>
    </div>

Note the underlined values in the code example above (175) would come from the CMS as actual data (current value), and need to all the same.

The same applies to the italicised values, but with the max value instead.
